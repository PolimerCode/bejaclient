# 🌊 BejaClient

**BejaClient** is a modern Minecraft client focused on performance, clean design, and a unique **water-inspired UI/UX experience**.

Built from the ground up with a custom launcher and modular architecture, BejaClient aims to deliver a premium experience similar to top-tier clients — while staying lightweight and flexible.

---

## ✨ Features

- ⚡ **High Performance**
  - Optimized rendering and minimal overhead
- 🎮 **PvP Focused**
  - Clean HUD, useful stats, competitive features
- 🌊 **Unique Water Theme**
  - Underwater-inspired UI (instead of typical space themes)
- 🧩 **Modular System**
  - Version adapters for different Minecraft versions
- 🚀 **Custom Launcher**
  - Built with Electron (HTML, CSS, JavaScript)
- 📊 **Profiles**
  - Switch between configs (PvP, SMP, etc.)
- 👥 **Friends System (planned)**
- 📰 **Integrated News (planned)**
